----------- Automatic and programmable kibble dispenser ---------------

This project was carried out as part of the C++ programming course given at INSA Toulouse. 

It consists of a mock-up of a kibble dispenser with the following functions:
  - Distribution of a dose of kibble from a web interface
  - Programming of an automatic distribution time
  - Indication of kibble level (via the interface or visually on the model)
More details in the "Report" file. 

The overall operation of the actions is in the application file. 
The project includes a configuration file indicating the pins used for the various sensors/actuators.  
Compilation: If the project does not compile, this is probably due to incorrect import of the libraries.

----------- Distributeur de croquette automatique et programmable ---------------

Ce projet à été réalisé dans le cadre du cours de programmation C++ dispensé à l'INSA Toulouse. 

Il s'agit d'une maquette d'un distributeur de croquette avec les fonctionnalités suivantes :
  - Distribution d'une dose de croquette depuis une interface web
  - Programmation d'une heure de distribution automatique
  - Indication du niveau de croquette (via l'interface ou visuellement sur la maquette)
Plus de détail dans le dossier "Rapport" fichier de compte rendu. 

Le fonctionnement global des actions se situe dans le fichier application. 
Le projet comporte un fichier configuration indiquant les pins utilisés pour les différents capteurs/actionneurs.  
Compilation : Si le projet ne compile pas cela est surement du à une mauvaise importation des bibliotheques.
